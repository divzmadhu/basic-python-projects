# Temperature Converter

> A beginner-friendly Python project that converts temperature between Celcius, Fahrenheit , and Kelvin.
---
## Features
- Accepts lowercase input
- Converts Celsius to Fahrenheit and Kelvin
- Converts Fahrenheit to Celsius and Kelvin
- Converts Kelvin to Celsius and Fahrenheit
---
## How to Run
1. Clone the repository:
   git clone https://github.com/<your-username>/temperature-converter.git
2. Navigate to the folder:
   cd temperature-converter
3. Run the program:
   python temperature_converter.py
---
## Sample Input/Output
**Input:**
```
Enter the temperature you want to convert: 273.15

Which unit do you want to convert from? (C/F/K): k
```

**Output:**
```
Celcius      Fahrenheit     Kelvin

0.00          32.00          273.15
```
