# Simple Calculator

> A beginner-friendly Python project that handles standard arithmetic operations: addition, subtraction, multiplication, and division. This project demonstrates fundamental Python function usage, control flow (if/elif/else), and input handling.
---
## Features
- Four Basic Operations: Supports +, −, ×, and ÷.
- Continuous Calculation: Allows the user to perform multiple calculations without restarting the program.
- Clean Interface: Uses a clear, menu-driven command-line interface.
- Accepts lowercase input while asking to continue the program
- Error Handling: division by zero.
---
## How to Run
1. Clone the repository:
   git clone https://github.com/divzmadhu/basic-python-projects/calculaor.py
2. Navigate to the folder:
   cd basic-python-projects/calculator
3. Run the program:
   python temperature_converter.py
