# Multiplication Table Generator

> A beginner-friendly Python project that will generate a complete multiplication table for a number specified by the user,
 up to a specified range. This exercise uses a for loop to repeatedly perform and print the multiplication operation.
---
## Features
* **Interactive Input:** Prompts the user to enter the base number and the multiplication limit.
* **Input Validation:** Ensures the specified limit is a positive number.
* **Looping Execution:** Allows the user to generate multiple tables without restarting the program.
* **Clear Output:** Displays the table in a clean, formatted manner.
---
## How to Run
1. Clone the repository:
   git clone https://github.com/divzmadhu/basic-python-projects.git
2. Navigate to the folder:
   cd multiplication-table
3. Run the program:
   python multiplication_table.py

## Sample Input/Output
**Input:**

```
Welcome to Multiplication Table Generator!
Enter the number for multiplication table: 7
Enter the limit: 5
```

**Output:**

```
7 * 3 = 21

Enter the number for multiplication table: 12

12 * 2 = 24
12 * 3 = 36
Do you want to continue(y/n): n
Exiting the program...
```
